## Sources footer

Every weekly file ends with two italicised lines, separated by a blank line:

1. **Sources line** — names the broad source families that the weekly draws on (NHS · Mayo · Cleveland Clinic · ACOG · RCOG · peer-reviewed maternal physiology · Evidence Based Birth) and links to the public repo where the per-domain provenance files live. This line is **mandatory** and **identical across all weeks** — do not vary it per week. The body of the file remains footnote-free (see `tone-guide.md`); the sources line is the single appendix-style citation, not inline references.
2. **Wikilink line** — `*Previous: [[Week-XX]] · Next: [[Week-XX]]*` — adapted per week per the rules below.

In `partner` and `pregnant` audience modes, the wikilinks use the mode-suffixed filenames (`Week-XX-partner`, `Week-XX-pregnant`); the Sources line is unchanged.

## Footer wikilink rules

- Week 4 (earliest): omit "Previous" — start with `*Next: [[Week-05]]*`
- Week 40 (latest): omit "Next" — `*Previous: [[Week-39]]*`
- All others: both present, separated by ` · `

## Header field rules

- **Size comparison**: Use a fruit/food/object that's roughly comparable. Common conventions: poppy seed (4w), apple seed (5w), sweet pea (6w), blueberry (7w), raspberry (8w), cherry (9w), strawberry (10w), lime (11w), plum (12w), peach (13w), lemon (14w), apple (15w), avocado (16w), turnip (17w), bell pepper (18w), heirloom tomato (19w), banana (20w), carrot (21w), papaya (22w), grapefruit (23w), corn cob (24w), cauliflower (25w), kale (26w), lettuce (27w), aubergine (28w), butternut squash (29w), cabbage (30w), coconut (31w), squash (32w), pineapple (33w), cantaloupe (34w), honeydew (35w), romaine (36w), winter melon (37w), pumpkin (38w), watermelon (39-40w).
- **Trimester boundaries**: First = weeks 1-13, Second = weeks 14-27, Third = weeks 28-40
- **Weeks to go**: 40 minus current week (rough — pregnancies vary)

## Section length guidance

Brief views (the part above each callout) should be scannable in 5-10 seconds. Detailed views can be 100-300 words. Total file should be 800-1500 words depending on the week — busier weeks (scans, milestones) get more, quieter weeks get less.

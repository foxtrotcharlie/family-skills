# Vittorio Protocol — Week-Indexed Reference

This file integrates the user's preferred protocol (from "How to Build a Human" by Vittorio) into weekly updates. Use it as a **source layer** for specific, week-anchored actions and stances. The voice of the weekly file stays warm and friend-like (see `tone-guide.md`) — do **not** adopt Vittorio's clipped/opinionated register. Translate his protocols into practical bullets that fit the existing template.

## How to use this file

When generating a week, scan the relevant phase below for items that anchor to that specific week. Pull them into the appropriate section of the weekly file:

- **Specific timed actions** (e.g. "perineal massage from week 34") → `This week's to-dos`
- **Symptom mechanism explanations** (e.g. why crackers worsen nausea) → `Mum's body` detailed breakdown
- **Partner support actions** (e.g. morning tray ritual) → `For your partner`
- **Decision points to flag for couple discussion** (e.g. vaccine choices) → to-dos under "Decisions"

If a Vittorio recommendation is contested or culture-specific (e.g. delayed Hep B, declined eye ointment, co-sleeping), present it as **a choice the couple should consciously make**, not as the default. See "Tone calibration" below.

## Phase 1 — Pre-conception (covered if user asks for pre-conception material)

The skill is week-numbered (4–40), so pre-conception isn't a typical request. If the user asks for "pre-conception" or "trying to conceive" content, the Vittorio protocol provides the spine:

- 74-day sperm maturation / 90-day egg-quality window — both partners should be in protocol 3 months before
- Folate 400–800 µg/day from now (methylfolate if MTHFR unknown)
- Quality prenatal: FullWell, Needed, or Thorne Basic — must hit 400+ mg choline
- Ferritin testing (target 30+, ideally 50+); blood volume expands 40–50% in pregnancy
- DHA 300–600 mg/day, third-party tested
- Eliminate endocrine disruptors (glass storage, no plastic heating, fragrance-free, EWG makeup)
- Both partners: alcohol near zero, sleep 7–8h, no testicle heating for him
- Full labs panels for both (see source for full list)
- Cycle tracking (Natural Cycles, TempDrop, BBT)
- The hard conversations: night wakes, career, help, family, village

## Phase 2 — First trimester anchors (weeks 4–13)

### The nausea protocol — primary content for weeks 6–12

**This is one of Vittorio's strongest mechanism arguments and the user values it.** Replace generic "eat crackers" advice in `Mum's body` and `For your partner` for these weeks with the protein-and-fat protocol:

- **Why crackers fail mechanistically:** Crackers spike glucose → insulin overshoots (pregnancy increases insulin resistance baseline so the overshoot is bigger) → glucose crashes → cortisol → more nausea. The advice creates the loop it claims to solve.
- **The rule:** Protein and fat before sleep, protein and fat first thing in the morning. Never let the body run empty overnight.
- **The nightly bowl:** Full-fat Greek yogurt + berries + drizzle of honey, before sleep.
- **The morning tray** (partner prepares before she's fully awake — rotation):
  - Sourdough toast + avocado + soft-scrambled eggs
  - Cottage cheese + prosciutto + cherry tomatoes
  - Parmesan + bresaola + figs
  - Soft-boiled egg + sardines on toast
  - Ricotta + walnuts + honey + pear
  - Smoked salmon + cream cheese + cucumber on rye
- **If nausea persists, escalate in order:** ginger 1 g/day → vitamin B6 25 mg 3x/day → Unisom (doxylamine) 12.5 mg + B6 at night (OTC Diclegis) → electrolyte hydration (LMNT, coconut water, salt+lime) → Sea-Bands at P6.
- **Hyperemesis escalation:** can't keep food/fluid down for 24h, weight loss, ketones in urine → medical, not white-knuckle. Zofran and IV exist.

### Choline emphasis (weeks 8 onward)

Standard prenatal delivers ~5% of fetal brain need. Three eggs/day covers most of the 450 mg target. If eggs aren't tolerated (common in first trimester), Jarrow Citicoline supplement.

### First trimester partner role (weeks 4–13)

- Maternal cortisol crosses the placenta. First-trimester cortisol affects fetal brain architecture in ways measurable years later. Frame partner's job as cortisol reduction, not just "being supportive."
- "Anything she needs, done already" baseline.
- Don't relitigate housework. Don't add emotional load.

## Phase 3 — Second trimester anchors (weeks 14–27)

### Specific week-anchored actions

- **Anatomy scan 18–22 weeks:** if soft markers flagged → second opinion before panicking. Ultrasound interpretation has variance.
- **Glucose screening 24–28 weeks:** if GD flagged, manage aggressively (low-carb, post-meal walks, glucometer). Changes birth risk profile.

### Body work that builds toward birth

- Daily walking 30–60 min
- Strength training 2–3x/week
- Hip mobility — deep squats opportunistically (e.g. while brushing teeth)
- Protein target 80–100 g/day (most women undershoot)
- Side-sleeping from mid-trimester (full-body U-shape pillow); supine compresses vena cava

### Partner reading list (to complete by week 30)

Partners often coast in T2 — use this window for the reading. Topics:

- Cord clamping timing

- Breastfeeding initiation and first 72 hours

- Vaccine decisions (Hep B at birth, vitamin K oral vs IM, eye ointment, infant schedule)
- Microbiome seeding (vaginal vs cesarean)
- Postpartum hormone cascade and mood

## Phase 4 — Third trimester anchors (weeks 28–40)

This is where Vittorio's protocol is most week-specific. Map directly onto weekly to-dos.

### Daily from week 28

- Walk 30–60 min
- Deep bodyweight squats — build to 30–60 sec holds
- Pelvic floor: 6–8 sec contract, full release, 8–12 reps × 3 sets/day. **Practice the relax** — that's what births the baby.
- Hip mobility nightly: pigeon, butterfly, hip circles on ball, cat-cow, malasana
- Belly breathing (diaphragmatic, expanding outward) until automatic

### Week-anchored to-dos

- **Week 27–36: Tdap vaccine.** Antibodies transfer to baby → ~78% pertussis risk reduction first 2–3 months. Vittorio kept this in his protocol despite delayed infant vaccination.
- **Week 34 onward: Perineal massage 3–4x/week, 5–10 min.** Natural oil (coconut, almond, vitamin E). Partner does it: thumbs 3–4 cm inside vaginal opening, firm downward U-shape pressure, hold stretch 1–2 min/spot. Reduces severe 3rd/4th-degree tears by 44%, suturing in first-time mothers by 9%.
- **Week 35–37: GBS test.** If positive → IV antibiotics during labor. Modern medicine working as intended.
- **Week 36 onward: Six dates per day (~70–75 g).** Evidence: arrived at hospital 3.5 cm dilated vs 2 cm, 96% spontaneous labor onset vs 79%, latent phase 4–7 hours shorter, less induction. Oxytocin-like compounds plus potassium/magnesium/tannins for smooth muscle. Cheap, safe, out-performs some pharmaceutical interventions.
- **Week 36 onward: Colostrum hand-expression 1–2x/day.** Collect into 1 mL oral syringes, cap, freeze. Build 30+ doses by birth. Insurance for difficult first hours, low blood sugar, delayed first latch. Safe from 36 weeks in low-risk pregnancies.
- **Hospital bag (pack ~36):** nursing bra, high-protein snacks (jerky, cheese, nuts — NOT granola bars), phone charger with **long** cable, pillow from home, maternity pads, baby outfit 0–3 months, soft blanket, car seat installed and rear-tested, **birth plan printed × 2**. Useless: books, laptop, entertainment.
- **Nursing stations (pre-birth):** 2–3 around the house — chair w/ armrests, water bottle, snacks, charger, burp cloths, nipple cream (lanolin or plant-based), arm pillow.
- **Stock the freezer.** Two weeks of meals minimum.

### Birth plan (weeks 28–36)

Three separate meetings with OB to walk through. Concede where evidence is thin, hold where strong. Get signatures: yours, OB's, midwife on shift. Bring two printed copies on the day. Worth flagging in to-dos around weeks 30–34.

Key positions in the birth plan worth surfacing:
- Freedom to move; no continuous monitoring unless clinically indicated; intermittent auscultation acceptable
- Eating and drinking freely
- Upright/side-lying delivery (not lithotomy) unless clinically indicated
- No episiotomy; no coached pushing; self-directed pushing on urge
- Delayed cord clamping 3+ min or until pulsation stops
- Immediate uninterrupted skin-to-skin minimum 60 min; assessments on mother's chest
- Newborn procedure preferences (oral vit K, declined eye ointment, deferred Hep B, deferred bath)
- Communication during labor goes through partner

## Phase 5 — Labor & delivery (use during labor or week ~40 if user asks)

### When to leave for hospital

- **3-1-1 (4-1-1 first-time):** contractions 3–4 min apart, 1 min long, for a full hour
- **Go immediately if:** waters break green/brown (meconium); bleeding beyond spotting; reduced fetal movement; suspected prolapsed cord; "something feels wrong" that won't resolve

### Early labor at home

- Eat — salty, protein, carbs (bone broth, sandwich, banana). Marathon, not sprint.
- Electrolytes
- Warm shower / dark room — oxytocin is shy (likes warm, dark, private)
- Stay off social media

### At hospital

- Hand over signed birth plan to midwife on shift
- State: communication goes through partner
- Refuse continuous monitoring unless indicated (intermittent auscultation 15–30 min is evidence-based)
- Refuse routine IV unless indicated — saline lock as compromise
- **Move:** walk, ball, hands and knees, side-lying, upright (reduces cesarean 29%)
- Counter-pressure on sacrum (heel of palm, hard, outward from spine — practice with a tennis ball weeks beforehand)

### Pain ladder

1. Warm water (bath/shower) — 30–40% pain reduction
2. Counter-pressure + breathing
3. Nitrous oxide if offered
4. Sterile water injections for severe back labor
5. Epidural as last resort (knows tradeoffs: mobility, pushing stage, instrumental delivery; it's a tool, not failure)

## Phase 6 — The golden hour (cover in week 40 or postpartum)

- Immediate skin-to-skin, baby on bare chest, blanket over both
- Delayed cord clamping 3+ min or until pulsation stops (~30% of total blood volume to baby)
- Door-sign rule: no unnecessary entries; assessments on mother's chest or deferred
- First latch within the hour — let baby self-attach (breast crawl is real)
- No eye ointment, no vit K injection, no bath, no weighing for first hour
- Partner: be present, still, quiet. Don't film. Don't post.

## Phase 7 — First 72 hours (postpartum content)

- Breastfeed every 1.5–3 hours, 8–12+ feeds/day; wake her if she sleeps >3 hours first week
- Hunger cues: rooting, hand-to-mouth, lip smacking, tongue thrust. **Crying is LATE.**
- Frozen colostrum syringes → finger or spoon feed (not bottle) in first 72h
- Skin-to-skin between every feed
- No pacifier first 4 weeks minimum; no bottle first 4–6 weeks unless medically necessary
- Hand expression first 3–5 days more effective than pump for colostrum
- **IBCLC threshold:** >10% birth weight loss, or below diaper counts → in the house within 24h. Credentials matter: International Board Certified Lactation Consultant, not "lactation counselor."

### Diaper count signal (first week)

- Day 1: 1+ wet, 1 black meconium
- Day 2: 2+ wet, 2 greenish stools
- Day 3: 3+ wet, 3 transitional (milk coming in)
- Day 4: 4+ wet, 3–4 yellow seedy
- Day 5+: 6+ wet, 3–4 yellow/day

## Phase 8 — Fourth trimester (weeks 1–12 postpartum)

### Sleep architecture

- **6 AM handoff:** partner takes baby off her chest into carrier for 2–3 hours. She gets second clean sleep block with eye mask.
- **Mother's sleep is the highest-value asset in the house.** Partner's job to protect at cost of his own.
- If exclusively breastfeeding, partner does NOT do night feeds — sleep, be functional in morning, don't perform wakefulness for solidarity.

### Skin-to-skin as medicine (postpartum)

- Body temp adjusts in real time
- Heartbeat (which baby already knows) stabilises baby's heart rate and breathing
- Cortisol buffer — measurable
- Partner skin-to-skin: oxytocin release real, attachment-relevant
- Carriers: BabyBjörn Mini (newborn) → BabyBjörn One (~3mo+)

### Postpartum supplement additions

Beyond pregnancy stack:
- Vitamin D3 6400 IU (discuss with provider — enriches breast milk, removes need for separate infant drops)
- Creatine 5 g AM + 5 g PM (Naked Creatine, pure monohydrate; sleep-deprivation cognition)
- Calcium to 1000 mg total

### Mental health screening

- Edinburgh Postnatal Depression Scale at 2 weeks; >12 → professional eval, don't wait for 6-week visit
- Baby blues (50–80%, days 2–3, resolves in 2 weeks) vs PPD (deepens, does NOT resolve)
- PPA: intrusive thoughts about harm to baby are common, OCD-spectrum, treat them, don't hide
- Partner is first line of screening — if something feels off, ask. Don't assume hormones.

### 6-week pelvic floor physio assessment

**Not optional.** Every woman should have one. Skipping it is how pelvic organ prolapse becomes "a normal part of aging." Worth flagging in the week 6 postpartum file or week 40 prep.

### Environment — first year

- No screens in baby's visual field for the first year
- No phones during feeding or holding (Still Face Experiment effect)
- No TV background as white noise
- Complex music daily (Bach, Mozart, Beethoven, Coltrane); live > recorded
- Glass bottles only (Philips Avent, Level 0 or 1 nipple); polypropylene leaches microplastic when heated

### Sleep setup

- Moses basket bedside first weeks
- If co-sleeping: **Safe Sleep Seven** — non-smoker · sober · breastfeeding · healthy full-term baby · baby on back · firm flat surface · no pillows/blankets/soft bedding near baby
- Sofa/recliner sleep is up to 67x more dangerous than bed; the "never co-sleep" message pushes mothers toward this. Plan for safe co-sleep, don't stumble into unsafe.

### Build the village

- Move closer to family if possible (single biggest predictor of smooth fourth trimester)
- Postpartum doula, night nurse, cleaner, meal delivery — infrastructure, not luxury
- "Say yes to" lists in to-dos when relevant: food drops, baby-holds while she showers
- "Say no to" first week: non-helping visitors, sick people, strong fragrance, recent long-haul travelers

## Phase 9 — Vaccine decision points

The user's protocol doesn't mean *all* readers should follow it, but the user has explicitly opted into this framing. When generating weekly content where vaccines arise, present as **decisions to make consciously**, not as defaults to accept or refuse:

### Defaults the user kept
- Maternal Tdap 27–36 weeks
- Measles 12–15 months (not 6 months — maternal antibodies still present)
- DTaP ~12 months (revisit per local exposure)
- BCG only if travel/exposure warrants

### Defaults the user declined
- Hep B at birth (if mother tested negative — discuss with care provider)
- Routine vit K IM (oral protocol instead — discuss)
- Erythromycin eye ointment (if clean STI panel)
- 2-month combined round (DTaP/Hib/PCV/IPV/RV/HepB)

### Triggers that flip "no" → "yes"
- Daycare enrollment
- International travel to endemic regions
- Immunocompromised family member in close contact
- Local outbreak

When this comes up in a weekly file, offer it as: "This is a decision worth making consciously with your pediatrician — many will default to the schedule without discussing alternatives. Ask them to walk you through the evidence for each one."

## Phase 10 — Solids (4–6 months postpartum, if generated)

### Readiness — all four required
- Sits upright with minimal support
- No tongue-thrust reflex
- Reaches for and grabs food
- Brings objects to mouth deliberately

### Start with
- Iron-first: pureed meat (beef, liver, lamb), lentils, egg yolk
- High-fat + high-protein: avocado, full-fat yogurt, cheese, eggs, meat
- Allergens early & often: peanut butter (thinned), egg, dairy, wheat, sesame, fish — one at a time, 3 days apart, in the morning
- Real food, baby-led weaning safely

### Skip
- Rice cereal as first food
- Fruit juice (zero)
- Added sugar (zero while you can hold the line)
- Cow's milk as beverage before 12 mo

## Tone calibration when integrating Vittorio

The Vittorio source is opinionated and direct. The weekly companion's voice is **warm, knowledgeable friend, not preachy.** Translation rules:

### Do

- Pull **specific actions** verbatim: "Six dates per day from week 36" lands clean.
- Pull **mechanism explanations** when they're genuinely useful (the nausea protocol's blood-sugar mechanism is one of the strongest examples).
- Use **week timing anchors** (Tdap 27–36, perineal massage from 34, GBS 35–37, anatomy scan 18–22, glucose 24–28, dates from 36, colostrum from 36) — these are gold for a weekly skill.
- Frame contested items as **decisions to make consciously**, e.g. "This is a decision worth discussing with your care provider — many couples default to X without realising they have a choice."

### Don't

- Don't import Vittorio's clipped/declarative voice ("declined", "we kept", "our recommendation") — translate into the existing warm-friend register
- Don't moralise about formula, screens, schedules — mention once, factually, no lecture
- Don't present declined items (Hep B at birth, eye ointment, 2-month round) as the recommended default — they're the user's personal choices, not universal advice
- Don't import the brand-locked product names (FullWell, Spectra S1, BabyBjörn, etc.) into every week — only when product context naturally arises in to-dos
- Don't import the macho-coded framing of the partner section ("the hierarchy", "what only she can do") — keep partner support warm and practical, not heroic
- Don't reproduce the protocol wholesale — surface the relevant slice for that week

### When Vittorio and existing weekly content disagree

- **Nausea management** (weeks 6–12): Vittorio's protein/fat protocol REPLACES the generic crackers advice. The user values this and the mechanism is sound.
- **Co-sleeping** (postpartum if generated): Lead with Safe Sleep Seven; don't repeat the "never co-sleep" line that the user explicitly disagrees with.
- **Hospital decisions** (week 40 / labor content): Lead with "decline X unless clinically indicated" framing rather than "follow hospital protocol."

### When the user hasn't signaled a stance

Default to existing weekly companion content. Use Vittorio as additive — not as a replacement when the topic isn't covered there. The skill remains the warm friend; Vittorio is the friend's reading list.

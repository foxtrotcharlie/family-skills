# family-skills

A small marketplace of [Claude Code](https://claude.com/claude-code) skills
for navigating pregnancy, newborn life, and the logistics of family. Skills
are grouped into independently-installable plugins so you only pull in what
you want.

## Install

```
/plugin marketplace add foxtrotcharlie/family-skills
/plugin install pregnancy@family-skills
```

## Plugins

### `pregnancy`

Pregnancy companion skills:

- **pregnancy-week-companion** — generates a single week's pregnancy update
  as an Obsidian-friendly markdown file. Six sections (baby, mum's body,
  partner support, what to say to her, to-dos, red flags) synthesised from
  NHS, Mayo, Cleveland Clinic, ACOG/RCOG, peer-reviewed maternal physiology
  papers, Evidence Based Birth, and curated diet/exercise/birth-prep briefs.
  Three audience modes: combined, partner-facing, or pregnant-person-facing.
  Mechanism in detail, lived experience in brief; options not prescriptions.

## Not medical advice

Everything these skills produce is general information, not medical advice.
Always consult your healthcare provider for anything specific to your pregnancy
or your family's health.
